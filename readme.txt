This plugin displays tooltips on the links widget and displays the user-defined link description.

== Description ==

This is a simple, lightweight plugin that adds tooltips to your 'links' widget using jQuery.  The tooltips appear with a bubble image and the description of the link.  please leave your comments if you have any suggestions for enhancement.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. Add a description to your links. 
2. Make sure you add the 'Links' widget to your sidebar.
3. Enjoy!
